Final Project: Smart Record Filing System

For my final project in PS70: Introduction to Digital Fabrication, I designed and built a Smart Record Filing System. This project combines physical fabrication techniques with electronic components and programming to create an intelligent solution for organizing and retrieving vinyl records.

Inspiration

The inspiration for this project came from Tolu Ademola's Fall 2024 project and my personal interest in vinyl records. I wanted to create a system that would not only store records efficiently but also help track and locate specific albums quickly.

How It Works

The Smart Record Filing System uses RFID tags to identify individual records. When placed on the turntable, the RFID reader located underneath sends the tag information to a microcontroller which manages the LED light system. The code recognizes multiple recurring signals as the music plays and switches to audio-visual. Once stopped it reverts back to storage mode.

Internals and Parts

The internal components include an ESP32-C6, RC522 RFID reader module, LED strip, and a custom-designed storage rack. All electronic components are housed inside the record player wich then controlls the LEDs inside the record shelf.
